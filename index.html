How it works

Three steps to an ensemble

Bagging (Bootstrap Aggregating) reduces variance by training independent models on different data samples and aggregating their decisions.

🎲

1. Bootstrap Sampling

For each tree, sample n rows from training data with replacement. Each sample is unique — some rows repeat, some are missing.

🌳

2. Train Decision Trees

Fit a full (or depth-limited) decision tree on each bootstrap sample. Trees are trained in isolation — no communication between them.

🗳️

3. Majority Vote

At prediction time, every tree votes for a class. The class with the most votes wins — this is majority voting aggregation.

The implementation

Three clean methods — bootstrap sampling, parallel tree training, and majority vote prediction. This is the exact pattern from the notebook.

import numpy as np
from collections import Counter
from sklearn.tree import DecisionTreeClassifier

class SimpleBaggingEnsemble:
    def __init__(self, n_trees=10, max_depth=None):
        self.n_trees  = n_trees
        self.max_depth = max_depth
        self.trees    = []

    # Sample n rows with replacement (bootstrap)
    def bootstrap_sample(self, X, y):
        n   = len(X)
        idx = np.random.choice(n, n, replace=True)
        return X[idx], y[idx]

    # Train one tree per bootstrap sample
    def fit(self, X, y):
        self.trees = []
        for _ in range(self.n_trees):
            X_s, y_s = self.bootstrap_sample(X, y)
            tree = DecisionTreeClassifier(max_depth=self.max_depth)
            tree.fit(X_s, y_s)
            self.trees.append(tree)

    # Majority vote across all trees
    def predict(self, X):
        preds = np.array([t.predict(X) for t in self.trees])
        return np.array([
            Counter(preds[:, i]).most_common(1)[0][0]
            for i in range(X.shape[0])
        ])
