# 🎯 Bagging Ensemble Learning Simulator

An interactive web application that demonstrates the **Bagging (Bootstrap Aggregating) Ensemble Machine Learning algorithm**. This project helps students and beginners understand how bagging works through visualization, simulation, and performance comparison.

## 📌 Features

- 📖 Introduction to Bagging Ensemble Learning
- 📊 Interactive Bagging Simulation
- 📈 Performance Comparison Charts
- 🎨 Responsive and Modern UI
- ⚡ Pure HTML, CSS, and JavaScript
- 📱 Mobile Friendly

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js

## 📖 What is Bagging?

Bagging (Bootstrap Aggregating) is an ensemble machine learning technique that improves model accuracy by training multiple models on different random subsets of the dataset and combining their predictions.

### Advantages

- Improves Accuracy
- Reduces Overfitting
- Stable Predictions
- Works with Decision Trees
- Easy to Implement
